## SEO Checklist for Articles

### Before Writing

- [ ] Research 5-10 related keywords
- [ ] Pick 1 primary keyword
- [ ] Pick 3-5 secondary keywords
- [ ] Check competitor articles
- [ ] Note internal linking opportunities

### While Writing

- [ ] Primary keyword in: title, H1, first 100 words
- [ ] Secondary keywords spread naturally
- [ ] H2 every 300-500 words
- [ ] Internal links to journeys/pages
- [ ] External links to authoritative sources
- [ ] Images with descriptive alt text

### After Writing

- [ ] Read aloud - does it flow?
- [ ] Check readability (aim for Grade 8 or below)
- [ ] Count words (minimum 800, ideal 1200+)
- [ ] Verify all internal links work
- [ ] Add FAQ section if not present

## Keyword Placement Guide

### Title Tag (H1)
```
Good: Sewa Private Yacht di Bali: Panduan Lengkap
Better: Sewa Yacht Bali: Panduan Lengkap untuk Charter Premium
```

### Meta Description
```
Format: [Primary keyword] + [Value proposition] + [CTA]
Example: Sewa yacht Bali mulai dari Rp 12.5 juta. Dapatkan pengalaman
private yacht premium ke Nusa Penida, Nusa Dua, dan Gili Islands. Booking sekarang!
```

### First Paragraph
```
Include: Primary keyword
Example: "Sewa yacht di Bali adalah pilihan terbaik untuk..."
```

### Headings (H2)
```
Format: [Action] + [Primary Keyword] + [Context]
Example:
- "Pilih Rute Yacht Berdasarkan Tujuan"
- "Kenapa Sewa Private Yacht di Bali?"
- "Tips Booking Yacht untuk First Timer"
```

### Alt Text Images
```
Format: [Primary keyword] + [Descriptive detail]
Example:
- "Private yacht di waters Bali dengan view Nusa Penida"
- "Sunset yacht charter di perairan Nusa Dua"
```

## Internal Linking Strategy

### Link to Journeys
```
Article about itinerary → Link to journey detail page
Article about proposal → Link to Nusa Dua sunset charter
Article about snorkeling → Link to Nusa Penida journey
```

### Link Structure
```
Homepage
├── /journeys/nusa-penida/
├── /journeys/nusa-dua/
├── /journeys/gili-trawangan/
└── /journal/[article-slug]/
    └── Link back to relevant journey
```

### Anchor Text Best Practices
```
Good: "Pelajari lebih lanjut tentang [yacht ke Nusa Penida]"
Better: "Lihat itinerary [Nusa Penida yacht trip] kami"

Bad: "Klik [disini]"
Better: "Hubungi [Marelune untuk booking]"
```

## FAQ Schema for SEO

Articles should end with FAQ section. Format:

```markdown
## FAQ: [Topic]

**Question 1?**
Answer 1...

**Question 2?**
Answer 2...
```

This generates "People Also Ask" rich snippets automatically.

## Quick Wins

### Easy Wins (Low competition)
```
- "[Location] yacht" (niche specific)
- "yacht untuk [occasion]" (birthday, proposal)
- "[duration] yacht trip" (3 hours, half day)
```

### High Intent Keywords
```
- "sewa yacht [location]" - ready to book
- "yacht charter [location] harga" - comparing prices
- "private yacht [location] near me" - high intent
```

### Long-tail Keywords
```
- "yacht untuk anniversary di bali"
- "sewa yacht 6 orang nusa penida"
- "private yacht untuk keluarga di bali"
```
